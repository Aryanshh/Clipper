# Clipper AI Video Editor

An automated platform that ingests video files or YouTube links, analyzes contents using Gemini 2.5, detects viral clips, and generates stylized captions.

## Features
- Dynamic 9:16 portrait video cropping
- Interactive visual subtitle editor
- High-retention active-word highlighting
- Direct YouTube Shorts metadata optimization card (Title, Description, Hashtags)
